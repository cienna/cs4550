User-first

As a new user, I need to be able to create an account.

Visit site

Enter personal info (name, username, password, email) in provided form fields

Click 'create account'

As an existing user, I need to be able to log in.

Visit site

Enter username/password info in available fields

Click ‘login’ button

As an existing user, I need to be able to follow/unfollow other users.

Visit another user’s page

Click ‘follow’ button

As an existing user, I need to be able to shout into the void.

Visit site

Go to your page

Type message in text box

Hit ‘submit’ button

As an existing user, I need to be able to delete poorly thought out posts.

Visit site

Go to your page

Go to the post you want to delete

Click ‘delete’ icon

Confirm you want to delete

As an existing user creating a post, I need to be able to mention other users.

Go to your page

Type a post, including @(username) for an existing user

Submit the post

As an existing user, I need to be able to search hashtags.

Go to site

Type desired hashtag in the search bar

Press enter or click the 'Do it' button

 

Data-first

User

Name

Username - unique, not changeable after account creation

Password

Email - unique

The most relevant page for this attribute is new/edit, which will be a form with type-in fields. Users won't be able to edit username after account creation, but everything else will be editable. An index page will show many users when one searches, but will display searched users' usernames, actual names, and most recent posts. Users will be able to go to an 'about' (show) page for other users, which will show their username, name, and email.

User

List of posts

List of followers

List of users they’re following

A profile page will show the posts this user has made in the center of the page, with modules or tabs off to the side to show users this user is following and being followed by.

Post

Message

Timestamp - system generated

User

The create post page will have one large text box and a subnit button. An index of posts will show when someone searches posts; many posts from many different users will appear. A timestamp will appear in an off color since it's not the focus, post content is. A show page will show one post's message, timestamp, and the name of the user who posted it. The user's name will link to their profile.

Follow

The following user

The user being followed

Follows won't have their own page on the site, but an index-like view will be a part of user's profiles. To follow or unfollow someone, a user will need to visit their page and click an icon.

Feed

List of posts

This is an index of posts, relevant either because a user is following the people who created the posts, or the user searched a term related to the posts.
